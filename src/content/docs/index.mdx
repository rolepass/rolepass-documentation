Rolepass replaces hand-managed IAM roles and long-lived access keys with a single
declarative config. Define your roles once, preview the exact changes, and apply
them to any number of AWS accounts; all trust is federated through GitHub or
GitLab OIDC, so no static credentials ever leave your CI system.
